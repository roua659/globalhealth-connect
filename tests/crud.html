API Endpoints
Publication Endpoints:
  • GET index.php?resource=publication&action=getAll - Get all publications
  • GET index.php?resource=publication&action=getById&id=1 - Get specific publication
  • POST index.php?resource=publication&action=create - Create publication
  • POST index.php?resource=publication&action=update&id=1 - Update publication
  • POST index.php?resource=publication&action=delete&id=1 - Delete publication
Comment Endpoints:
  • GET index.php?resource=commentaire&action=getAll - Get all comments
  • POST index.php?resource=commentaire&action=create - Create comment
  • POST index.php?resource=commentaire&action=delete&id=1 - Delete comment
